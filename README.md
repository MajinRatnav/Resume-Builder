# Résumé Building Application

A robust Java-based desktop utility designed to automate the creation of professional resumes. [cite_start]By streamlining the multi-step input process, this application reduces manual formatting time by approximately 80%[cite: 28, 30, 33].

## 🚀 Features

* [cite_start]**Automated PDF Generation**: Leverages the **iTextPDF library** to programmatically transform structured user input into industry-standard PDF documents[cite: 31].
* [cite_start]**Intuitive UI/UX**: Built with a custom **Java Swing** interface to collect complex professional data—including education (CGPA, 10th/12th marks), work experience, and projects—through a streamlined workflow[cite: 28, 33].
* [cite_start]**Automated File Management**: Includes robust file-handling logic that automatically organizes and stores generated resumes within the root directory for easy access[cite: 32].
* **Professional Formatting**: Ensures consistent layout, font scaling, and section headers (Education, Skills, Experience) to meet modern recruitment standards.

## 🛠️ Tech Stack

* [cite_start]**Language**: Java [cite: 28]
* [cite_start]**GUI Framework**: Java Swing / JavaFX [cite: 30]
* [cite_start]**PDF Engine**: iTextPDF Library [cite: 31]
* [cite_start]**IDE**: Eclipse [cite: 28]

## 📋 Data Points Captured

The application captures comprehensive professional profiles, including:
* **Personal Info**: Name, Email, Phone, and Address.
* **Academic History**: Degree, CGPA, 12th Marks, and 10th Marks.
* [cite_start]**Professional Background**: Years of Work Experience, Technical Skills, and Project Details[cite: 6, 9].
* **Personal Interests**: Hobbies and extra-curricular activities.

## 🔧 Installation & Setup

1.  **Clone the Repository**:
    ```bash
    git clone [https://github.com/MajinRatnav/Resume-Builder.git](https://github.com/MajinRatnav/Resume-Builder.git)
    ```
2.  **Add iText Dependencies**: 
    [cite_start]Ensure the iText 7 (kernel, io, and layout) JAR files are added to your project build path in Eclipse[cite: 28].
3.  **Run the Application**:
    Execute the `ResumeBuilder1.java` file to launch the GUI.

## 📖 Usage

1.  Enter your professional and academic details into the provided fields.
2.  Click the **"Generate PDF Resume"** button.
3.  Choose your desired save location via the file chooser.
4.  The application will instantly generate a formatted PDF ready for job applications.
